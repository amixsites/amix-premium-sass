# Amix Premium SaaS Agency Website

A world-class, light-themed SaaS agency portfolio website built with Next.js, React, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Premium Light Theme** — Clean, modern aesthetic with glassmorphism effects
- **Framer Motion Animations** — Smooth scroll-triggered reveals, floating elements, and micro-interactions
- **Fully Responsive** — Mobile-first design with optimized layouts for all screen sizes
- **SEO Optimized** — Meta tags, semantic HTML, and performance-focused architecture
- **All Original Content Preserved** — School ERP, Restaurant POS, Banquet Manager, E-Commerce, Business Websites
- **Interactive Components** — Animated counters, carousels, magnetic buttons, hover effects
- **Production Ready** — Optimized for Vercel deployment with static export support

## Sections

1. **Hero** — Animated headline, floating dashboard mockups, metrics counters
2. **Services** — 5 core service cards with feature lists
3. **Statistics** — Animated counters on scroll
4. **Projects** — CutieBox, Maison Rose, Shrikha Organics with live demos
5. **SaaS Products** — Dashboard mockups for School ERP, POS, Banquet Manager
6. **Process** — 7-step timeline with scroll-triggered reveals
7. **Testimonials** — Auto-scrolling carousel with glass cards
8. **Tech Stack** — Animated technology grid with marquee
9. **Why Amix** — 6 advantage cards
10. **Contact** — Interactive form + WhatsApp/Email/Phone CTAs
11. **Footer** — Multi-column links with social icons

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS 3.4
- Framer Motion
- Lucide React

## Deployment

### Vercel (Recommended)

1. Push code to GitHub
2. Import repository on [Vercel](https://vercel.com)
3. Deploy with default settings

### Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```
